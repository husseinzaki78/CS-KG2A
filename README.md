# KG2-A Parent Communication System

نظام إدارة أولياء أمور فصل KG2-A مع Firebase

## 🚀 رفع المشروع على GitHub

### الخطوة 1: إنشاء Repository جديد
1. اذهبي إلى [GitHub](https://github.com)
2. اضغطي **New repository** (الزر الأخضر)
3. اسم الـ Repository: `kg2a-system` (أو أي اسم تحبيه)
4. اختاري **Public**
5. اضغطي **Create repository**

### الخطوة 2: رفع الملف
**طريقة سهلة (من الموقع):**
1. في صفحة الـ Repository الجديد
2. اضغطي **uploading an existing file**
3. اسحبي ملف `kg2a_system_v2.html`
4. غيري الاسم لـ **index.html** (مهم!)
5. اضغطي **Commit changes**

### الخطوة 3: تفعيل GitHub Pages
1. في الـ Repository، اضغطي **Settings** (آخر تاب)
2. من القائمة الجانبية: **Pages**
3. تحت **Source**:
   - اختاري **Deploy from a branch**
   - Branch: **main** (أو **master**)
   - Folder: **/ (root)**
4. اضغطي **Save**
5. انتظري دقيقة أو اتنين
6. هيظهر لينك زي: `https://username.github.io/kg2a-system/`

### 🎉 خلاص!
دلوقتي السيستم متاح من أي جهاز عن طريق اللينك ده!

## 📱 كيفية الاستخدام

### 1️⃣ فتح السيستم
افتحي اللينك من أي موبايل أو كمبيوتر واحفظيه في المفضلة

### 2️⃣ إرسال إيميلات فردية مخصصة
1. تبويب **📝 القوالب**
2. اضغطي **✉️ استخدام** على القالب اللي عايزاه
3. هيفتح تبويب الإيميل ويختار كل الطلاب
4. اضغطي **✨ إرسال إيميلات مخصصة**
5. هتفتح قائمة كل طالب له إيميل خاص باسمه
6. اضغطي **📧 إرسال** لكل طالب على حدة

### 3️⃣ المتغيرات في القوالب
استخدمي في القوالب:
- `STUDENT_NAME` أو `[اسم الطالب]` → يتحول لاسم الطالب
- `STUDENT_ID` أو `[رقم الطالب]` → يتحول لرقم الطالب

مثال:
```
Dear Parents,
We are proud of STUDENT_NAME for their achievements...
```

### 4️⃣ مشاركة البيانات
- كل التعديلات تحفظ تلقائياً على Firebase
- أي حد يفتح اللينك هيشوف نفس البيانات
- التحديثات التلقائية كل 30 ثانية

## 🔐 إعدادات Firebase (مهم!)

تأكدي إن Firebase Rules مظبوطة:
1. [Firebase Console](https://console.firebase.google.com)
2. المشروع بتاعك → **Realtime Database** → **Rules**
3. غيري القواعد لـ:
```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```
4. **Publish**
